# 欢迎来到我的博客

这是一个基于 **Next.js** 和 **MDX** 构建的现代化个人博客。

## 特性

### 📝 易于写作
- 使用 Markdown 语法编写文章
- 支持 MDX，可以在 Markdown 中使用 React 组件
- 通过 Frontmatter 管理文章元数据

### 🚀 性能优化
- 静态生成（SSG），页面加载速度极快
- 自动代码分割
- SEO 友好

### 🎨 现代化设计
- 响应式布局
- 代码高亮
- 优雅的排版

## 开始写作

在 `content/posts` 目录下创建 `.mdx` 或 `.md` 文件即可开始写作！

支持以下 Frontmatter 字段：

- `title`: 文章标题
- `date`: 发布日期（ISO 格式）
- `excerpt`: 文章摘要
- `tags`: 标签数组
- `author`: 作者名称

## 技术栈

- **Next.js 16**: React 框架
- **MDX**: Markdown + JSX
- **TypeScript**: 类型安全
- **Tailwind CSS**: 样式框架

祝你写作愉快！🎉
